# Usage

1) Method 1
- Open Terminal
- cd Admin-panel
- npm run build
- Copy all files in admin-panel/build to backend/public 
- Open Terminal
- cd backend
- npm start
2) Method 2
- Open Terminal
- cd backend
- npm start
- Open Terminal
- cd Admin-panel
- npm run start
